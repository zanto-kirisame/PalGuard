# PalGuard

パルワールド (Palworld) のパッチ型MOD (.pak) および UE4SS スクリプトを管理するための軽量・高速なツールです。

## 主な機能

- **MODの有効・無効切り替え**: クリック一つでMODを切り替えます。
- **優先順位（ロード順）設定**: 直感的な上下ボタンで、競合するMODの優先順位を調整します。
- **軽量なファイル競合検知**: 外部ツールを使わず、`.pak` ファイルの内部インデックスを直接解析。特定のアセット（例：特定のパルのデータ）が重複して上書きされるのを検知します。
- **競合状態の可視化**: どのMODがどのアセットを上書きしているか、ロード順に基づいた「勝者」を一目で判断できます。
- **外部ツールの更新監視**: Palworld本体、UE4SS、Pal Schemaの最新バージョンを自動チェック。更新通知とバックアップの重要性をリマインドします。
- **UIの安定化**: タブ切り替え時のガタつきを修正し、よりスムーズな操作感を実現。
- **多言語対応**: 日本語と英語に対応しています。

## インストール方法

1. [配布ページ] から `PalGuard_v1.2.0.zip` をダウンロードします。
2. 好きな場所に解凍します。
3. `PalGuard.exe` を実行します。
4. 設定画面から、パルワールドのインストールフォルダを選択してください。
   - 例: `C:\Program Files (x86)\Steam\steamapps\common\Palworld`

## 使用方法

### MOD管理
- 「有効」スイッチでMODを有効にします。
- リストの上下ボタンを使ってMODの順番を入れ替えることで、同じアセットを修正するMODの優先順位（どちらで上書きするか）を決定できます。
- **競合チェック**: PalGuardは自動的に `.pak` 内をスキャンします。複数のMODが同じファイル（例：`Character/Pals/BP_PalStatus.uasset`）を書き換えようとしている場合、警告が表示されます。

### ログ解析
- アプリがクラッシュしたり、MODが反映されない場合は「ログ解析」タブを確認してください。
- 既知のエラーパターンから原因を推測し、対策を表示します。

## 重要：免責事項とバックアップ

> [!WARNING]
> **利用は自己責任で行ってください。**
> PalGuardはゲームファイルを直接削除することはありませんが、ファイル名の変更を伴う操作を行います。
> MODを導入・管理する前に、以下のフォルダのバックアップを強く推奨します。
> `.../Pal/Content/Paks`
