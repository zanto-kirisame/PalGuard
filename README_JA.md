# PalGuard

パルワールド (Palworld) のパッチ型MOD (.pak) および UE4SS スクリプトを管理するための軽量・高速なツールです。

## 主な機能

- **MODの有効・無効切り替え**: クリック一つでMODを切り替えます。
- **優先順位（ロード順）設定**: 直感的なドラッグ＆ドロップ（または上下ボタン）で、競合するMODの優先順位を調整します。
- **競合状態の可視化**: どのMODがどのアセットを上書きしているか、どちらが「勝者」かを一目で判断できます。
- **高度なログ解析**: `Pal-Trace.log` や `UE4SS.log` をスキャンし、クラッシュ等の原因を日本語で解説。解決のヒントを提示します。
- **多言語対応**: 日本語と英語に対応しています。

## インストール方法

1. [配布ページ] から `PalGuard_v1.0.0.zip` をダウンロードします。
2. 好きな場所に解凍します。
3. `PalGuard.exe` を実行します。
4. 設定画面から、パルワールドのインストールフォルダを選択してください。
   - 例: `C:\Program Files (x86)\Steam\steamapps\common\Palworld`

## 使用方法

### MOD管理
- 「有効」スイッチでMODを有効にします。
- リストの上下ボタンを使ってMODの順番を入れ替えることで、同じアセットを修正するMODの優先順位（どちらで上書きするか）を決定できます。

### ログ解析
- アプリがクラッシュしたり、MODが反映されない場合は「ログ解析」タブを確認してください。
- 既知のエラーパターンから原因を推測し、対策を表示します。
